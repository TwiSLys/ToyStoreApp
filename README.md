# ООО «МирИгрушек»

Настольная информационная система для просмотра товаров и управления
товарами и заказами. Приложение разработано на Python, Tkinter и SQLite.

## Запуск готовой версии

Откройте папку `dist\ToyStore` и запустите `ToyStore.exe`.
База данных и ресурсы должны оставаться рядом с исполняемым файлом.

## Возможности

- гость и клиент: просмотр каталога;
- менеджер: поиск, фильтрация, сортировка, просмотр заказов;
- администратор: все возможности менеджера, добавление, изменение и удаление
  товаров и заказов;
- подсветка товаров без остатка и товаров со скидкой более 17%;
- расчет цены со скидкой;
- хранение изображений размером не более 300 x 200 пикселей;
- запрет удаления товара, который присутствует в заказе;
- проверка полей и информативные сообщения об ошибках.

## Запуск из исходного кода

```powershell
python -m pip install -r requirements.txt
python build_database.py
python app.py
```
